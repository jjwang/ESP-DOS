# OpenCrab-DOS 硬件接线指南

## 引脚总览

| 外设 | 接口 | GPIO | 说明 |
|------|------|------|------|
| ST7789 显示屏 | SPI2 | CS=10, DC=11, RST=1, MOSI=13, SCLK=12, BL=14 | 320×170 |
| TCA8418 键盘 | I2C_NUM_1 | SDA=4, SCL=5 | 地址 0x34 |
| DS3231 RTC | I2C_NUM_0 | SDA=15, SCL=16 | 地址 0x68 |

---

## ST7789 显示屏

| ST7789 | ESP32-S3 |
|--------|----------|
| CS     | GPIO 10  |
| DC     | GPIO 11  |
| RST    | GPIO 1   |
| MOSI   | GPIO 13  |
| SCLK   | GPIO 12  |
| BL     | GPIO 14  |
| VCC    | 3.3V     |
| GND    | GND      |

**配置**: SPI2 (SPI_HOST), 40MHz, 列偏移 35 (170px 居中), MADCTL=0x60 (横屏)

---

## TCA8418 键盘

**芯片**: Texas Instruments TCA8418（非 NXP TCA8418E）

| TCA8418 | ESP32-S3 |
|---------|----------|
| SDA     | GPIO 4   |
| SCL     | GPIO 5   |
| VCC     | 3.3V     |
| GND     | GND      |

### 4×4 矩阵键盘

| 键盘行 | TCA8418 |
|--------|---------|
| 物理 R0 (1 2 3 A) | R3 |
| 物理 R1 (4 5 6 B) | R2 |
| 物理 R2 (7 8 9 C) | R1 |
| 物理 R3 (* 0 # D) | R0 |

| 键盘列 | TCA8418 |
|--------|---------|
| C0-C3  | C0-C3 (直连) |

### 键位布局

```
 ┌─────┬─────┬─────┬─────┐
 │  1  │  2  │  3  │  A  │  ← 物理 R0 (TCA R3)
 ├─────┼─────┼─────┼─────┤
 │  4  │  5  │  6  │  B  │  ← 物理 R1 (TCA R2)
 ├─────┼─────┼─────┼─────┤
 │  7  │  8  │  9  │  C  │  ← 物理 R2 (TCA R1)
 ├─────┼─────┼─────┼─────┤
 │  *  │  0  │  #  │  D  │  ← 物理 R3 (TCA R0)
 └─────┴─────┴─────┴─────┘
```

## DS3231 RTC

| DS3231 | ESP32-S3 |
|--------|----------|
| SDA    | GPIO 15  |
| SCL    | GPIO 16  |
| VCC    | 3.3V     |
| GND    | GND      |

系统启动时自动从 DS3231 读取时间同步到 ESP32 系统时钟。
可用 `DATE` 和 `TIME` 命令查看/设置。
