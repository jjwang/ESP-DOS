# TCA8418 键盘 & DS3231 RTC 接线指南

## I2C 总线分配

| 外设 | I2C 控制器 | SDA | SCL | 地址 |
|------|-----------|-----|-----|------|
| TCA8418 键盘 | I2C_NUM_1 | GPIO 4 | GPIO 5 | 0x34 |
| DS3231 RTC | I2C_NUM_0 | GPIO 15 | GPIO 16 | 0x68 |

两个设备使用独立的 I2C 控制器，引脚互不冲突。

## TCA8418 键盘

**芯片**: Texas Instruments TCA8418（非 NXP TCA8418E）

| TCA8418 | ESP32-S3 |
|---------|----------|
| SDA | GPIO 4 |
| SCL | GPIO 5 |
| VCC | 3.3V |
| GND | GND |

### 4×4 矩阵键盘

| 键盘行 | TCA8418 |
|--------|---------|
| 物理 R0 (1 2 3 A) | R3 |
| 物理 R1 (4 5 6 B) | R2 |
| 物理 R2 (7 8 9 C) | R1 |
| 物理 R3 (* 0 # D) | R0 |

| 键盘列 | TCA8418 |
|--------|---------|
| C0-C3 | C0-C3 (直连) |

### 键位布局

```
 ┌─────┬─────┬─────┬─────┐
 │  1  │  2  │  3  │  A  │  ← 物理 R0 (TCA R3)
 ├─────┼─────┼─────┼─────┤
 │  4  │  5  │  6  │  B  │  ← 物理 R1 (TCA R2)
 ├─────┼─────┼─────┼─────┤
 │  7  │  8  │  9  │  C  │  ← 物理 R2 (TCA R1)
 ├─────┼─────┼─────┼─────┤
 │  *  │  0  │  #  │  D  │  ← 物理 R3 (TCA R0)
 └─────┴─────┴─────┴─────┘
```

## DS3231 RTC

| DS3231 | ESP32-S3 |
|--------|----------|
| SDA | GPIO 15 |
| SCL | GPIO 16 |
| VCC | 3.3V |
| GND | GND |

系统启动时自动从 DS3231 读取时间同步到 ESP32 系统时钟。
